# Pilot compiled dataset — Tumbiana (pre-GOE) vs Gunflint (post-GOE)

**Compiled:** 2026-07-12. **Rule:** every entry is sourced to a real publication; "?" marks a
value not found in the open literature during this compilation (reported as a gap, not estimated).

## 1. Formation attributes

| Attribute | Tumbiana Formation (pre-GOE) | Gunflint Formation (post-GOE) | Source |
|---|---|---|---|
| Age | 2724 ± 5 Ma (2.72 Ga) | ~1.88 Ga | Lepot2009; Lepot2017 |
| Basin / group | Hamersley Basin, Fortescue Gp., Pilbara Craton | Animikie Basin, Ontario | Lepot2009; Barghoorn&Tyler1965 |
| Lithofacies | stromatolitic carbonate + chert; shallow-water / lacustrine (Meentheena Mbr) | stromatolitic chert ("algal chert member"); shallow, wave-influenced, ferruginous | Lepot2009; Lepot2017 |
| Depositional redox (qualitative) | shallow, possible oxygenic-photosynthesis haven; complex local S-cycling | ferruginous, **sulfate-limited** | Eigenbrode&Freeman2006 (via Lepot2009); Wacey2013, Planavsky2009 (via Lepot2017) |
| **Thermal maturity (qualitative)** | "weakly metamorphosed"; low-grade **<300 °C** | low-grade, **prehnite–pumpellyite facies**; Schreiber Beach = least metamorphosed | Decreane2021; Floran&Papike1975, Alleon2016 (via Lepot2017) |
| **Thermal maturity (RSCM R2 / Tmax)** | **? — not found as a published RSCM T** (constraint is mineralogical, <300 °C) | documented via Raman + synchrotron as a function of T | Lepot2009 (Raman used for CM structure, not RSCM T); Alleon2016 |

## 2. Organic matter / preservation

| Attribute | Tumbiana | Gunflint | Source |
|---|---|---|---|
| OM abundance | abundant organic globules (larger OM pool than the rare microfossils) | dense microfossil populations; kerogen-rich laminae | Lepot2009; Lepot2017 |
| OM character | two pools: Type A (S-poor, silicate-associated) + Type B (S-rich, thiophenic globules in calcite) | kerogenous microfossil walls; amide/protein-derived molecules preserved at Schreiber | Lepot2009; Alleon2016 (via Lepot2017) |
| Morphological microfossils | **rare** ("a few putative filamentous microfossils"); globules are cell-like but not diagnostic | **diverse & diagnostic** (Gunflintia minuta Types 1/2, Huroniospora thin/thick-walled, Animikiea, G. grandis) | Lepot2009; Lepot2017 |
| δ¹³C org | extremely ¹³C-depleted (methanotrophy attribution) | per-microfossil consistent with oxygenic photosynthesis / heterotrophy | Hayes1994, Thomazo2009 (via Lepot2009); House2000, Williford2013 (via Lepot2017) |

## 5. Data gaps identified (honest)

1. **No published RSCM peak-T (R2/Tmax) for Tumbiana** — only a mineralogical <300 °C constraint. A quantitative maturity **match** therefore cannot yet be demonstrated from published data; needs new RSCM on Tumbiana.
2. **No bulk Poulton–Canfield FeHR/FeT or Fepy/FeHR for either unit** — existing Fe work is nanoscale mineralogy/isotopes (highly informative, but not the standard redox-proxy ratios). Bulk Fe-speciation comparison needs new measurements or a paired-shale redesign.
3. **Gunflint itself is locally sulfate-limited** (Wacey2013) despite post-GOE oceanic sulfate rise — a complication for any "post-GOE = sulfate-rich = pyritisation" logic.
