# Pilot — Tumbiana (pre-GOE) vs Gunflint (post-GOE)

Literature-data pilot for the GEODISC GOE-preservation framework (Assignment 1, "Seb").
Addresses the peer-review feasibility concern by compiling **real published** Raman / Fe /
preservation data for one matched pre-/post-GOE pair.

## Contents
- `compiled_dataset.md` — the compiled dataset: formation attributes (age, facies, redox,
  thermal maturity), OM/preservation, and Fe/sulfide/mineralization, each entry sourced. Includes
  an explicit "data gaps" section.
- `pilot_report.md` — analysis: feasibility finding, preliminary signal (Prediction 1), and a
  **mechanism-revision finding** (the framework's sulfate→pyritisation mechanism is not supported
  by this pair; silicification + Fe-biomineralization are the preservative pathways).
- `data/`, `figures/` — reserved for new measurements / figures in later phases.

## Rule
Fiction-free: every datum is sourced to a real publication; unfound values are reported as gaps
("?"), never estimated. See `compiled_dataset.md` §5.
